Detailed Description

Simple untrusted TCP application used by attacker to send garbage payloads.

Simulates a simple TCP client application that sends a message to a given peer.

A simple TCP-based application to send data to a remote peer.

This application is used by both the legitimate sensor and the spoofed attacker to transmit a message to the gateway node.
